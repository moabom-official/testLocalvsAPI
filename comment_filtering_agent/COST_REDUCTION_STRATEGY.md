## 💡 비용 절감 전략 (5단계)

### 전략 1: 1차 규칙 필터 강화 (즉시)

**목표**: LLM 호출 전에 70% 제거  
**비용**: $0  
**효과**: 기본 품질 유지, LLM 부하 감소  

```python
# 현재: 100,000 댓글 → 100,000 LLM 호출
# 1차 필터 강화 후: 100,000 댓글 → 30,000 LLM 호출

월간 절감: $350/월 (70% 감소)
```

**구현**:
- [ ] 규칙 추가 (반응 패턴, 인사말 등)
- [ ] 욕설 사전 보강
- [ ] 중복 댓글 감지

### 전략 2: 배치 처리 최적화 (1개월)

**목표**: LLM API 호출 횟수 감소  
**비용**: $0  
**효과**: 처리 속도 향상, 비용 동일  

```python
# Before: 댓글 1개씩 API 호출
for comment in comments:
    result = api_call(comment)  # 1,000회

# After: 배치 처리
batch_size = 50
for batch in chunks(comments, batch_size):
    results = api_call_batch(batch)  # 20회

API 호출: 1,000회 → 20회
처리 속도: 3배 향상
```

**구현**:
- [ ] LLM API 배치 엔드포인트 사용
- [ ] 파이프라인 배치 크기 최적화

### 전략 3: Confidence Threshold 조정 (3개월)

**목표**: Agent에서 재검증 줄이기  
**비용**: $0  
**효과**: Confidence 분포 분석 후 threshold 최적화  

```python

high_conf (≥0.8): 85% → 바로 사용
medium_conf (0.6-0.8): 10% → 재검증 필요?
low_conf (<0.6): 5% → 재검증

# Threshold 조정으로 재검증 50% 감소
재검증 비용: $50/월 → $25/월
```

**구현**:
- [ ] Confidence 분포 모니터링
- [ ] 최적 Threshold 찾기 (A/B 테스트)
