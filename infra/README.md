# Azure 배포

Moabom 프로토타입을 Azure 에 배포하기 위한 IaC 와 스크립트.

## 구성

| 리소스 | 역할 | 월 예상 비용 (USD) |
|---|---|---|
| Container Registry (Basic) | Docker 이미지 저장 | ~$5 |
| Postgres Flexible (B1ms, 32GB) | DB | ~$13 |
| Container Apps (Consumption) | FastAPI 앱, scale-to-zero | $0~5 |
| Log Analytics | 로그 | 사용량 비례 |
| **합계** |  | **~$20/월** |

## 파일

- `main.bicep` — 인프라 정의 (ACR / Postgres / Log Analytics / Container Apps Env)
- `deploy.sh` / `deploy.ps1` — bash / PowerShell 각 버전. 동일 동작.
- `teardown.sh` / `teardown.ps1` — 전체 리소스 삭제 (과금 중지)

## 배포 순서 (Git Bash / WSL)

```bash
# 1. Azure CLI 설치 확인
az --version

# 2. 로그인 (브라우저 열림)
az login

# 3. subscription 선택
az account list --output table
az account set --subscription "<subscription-id-or-name>"

# 4. 배포 실행 (Postgres 비밀번호 입력 프롬프트)
./infra/deploy.sh
```

PowerShell 사용 시: `./infra/deploy.sh` 를 `.\infra\deploy.ps1` 로 대체.

완료되면 `https://<auto-generated>.koreacentral.azurecontainerapps.io` URL 이 출력됩니다.

## 재배포 (코드 변경 후)

```bash
./infra/deploy.sh
```

같은 스크립트를 다시 돌리면 새 이미지 태그로 빌드 후 Container App 이 롤링 업데이트됩니다.

## 로그 확인

```bash
az containerapp logs show -n ca-moabom -g rg-moabom --follow
```

## 비용 절감 팁

- 데모가 끝나면 `teardown.sh` 실행 → 월 비용 $0
- Postgres 를 끄고 싶지만 지우기 싫다면: `az postgres flexible-server stop -n <서버명> -g rg-moabom`
  (최대 7일까지 중지 가능, 이후 자동 재시작)

## 데이터 초기 상태

앱이 첫 요청을 처리하기 전 `main.py` 의 `startup_event()` 가 `init_db()` 를 호출해 테이블과 인덱스를 자동 생성합니다. 별도의 마이그레이션 단계 불필요.
